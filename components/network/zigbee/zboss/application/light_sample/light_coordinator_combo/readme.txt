Light Coordinator Combo application:
====================================
Light Coordinator Combo device is based on the light_coordinator application which is operated also as ZGP Sink. The device
can join ZGP devices to the network, also supports ZGP Auto-commissioning mode. The Light Coordinator Combo can retransmit
commands from ZGPD to another device. For example, a bulb joined to the Light Coordinator Combo device can be toggled by ZGPD Switch.


Setup for demo with ZGPD Switch:
================================
The demo should be set in the following way:
1. Flash each device individually and power-off after programming.
2. Power on Light Coordinator Combo.
3. Power on Dimmable Light device.
4. Power on ZGPD Switch.
