Color Light ZED sample
---------------------------------------
This application is Zigbee 3.0 End Device with Color Light functionality.
Supported clusters: Identify, On/Off, Level Control, Color Control, Groups, Scenes.
Default operational channel is 11.

Use case:
1. Start ZB coordinator device (LCGW, for example), turn on assotiation permit.
2. Join Color Light device to the coordinator.
3. Control On/Off, Level values of the lamp etc.
