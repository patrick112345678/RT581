13.10 TP/R22/SGMB-10 Test case for Unknown and Broadcast Commands

Objective:

Verify various unsupported and broadcast command behaviour.
1. Unsupported Unicast ZDO commands shall be answered while unsupported
broadcast ZDO commands are not.

2. Unsupported Unicast NWK commands are answered while unsupported broadcast
NWK commands are not.

3. Unicast ZDO Binding commands are accepted while broadcast ZDO binding
commands are not.
