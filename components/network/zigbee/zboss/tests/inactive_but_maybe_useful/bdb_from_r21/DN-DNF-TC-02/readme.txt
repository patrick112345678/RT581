2.1.2	FS-PRE-TC-02: Channel set - association test
This test verifies the DUT's ability to join an open network on any channel of DUT's bdbPrimaryChannelSet and bdbSecondaryChannelSet.
The device under test takes the role described in its PICS; the other role is performed by a TH router.
Successful completion of this test is pre-condition for performing any formation and steering tests from section 3.3-3.5 and 4.2- 4.5.
