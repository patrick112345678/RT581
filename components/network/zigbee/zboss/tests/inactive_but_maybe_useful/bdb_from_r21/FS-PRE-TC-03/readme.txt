2.1.3	FS-PRE-TC-03: Channel scanning as part of network formation
This test verifies the channel scanning behavior of the base device during network formation procedure. It verifies the correct usage of the bdbPrimaryChannelSet (and bdbSecondaryChannelSet).
The device takes the role described in its PICS; no other devices are needed.
Successful completion of this test is pre-condition for performing any relevant formation tests from section 3.3-3.5 and 4.2- 4.5.
